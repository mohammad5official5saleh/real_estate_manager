# User Stories

## Authentication
- As a user, I can create an account.
- As a user, I can log in to my account.
- As a user, I can log out securely.

## Property Management
- As a user, I can add a property.
- As a user, I can edit my property.
- As a user, I can delete my property.
- As a user, I can view my own properties.

## Property Viewing
- As a visitor, I can view available properties.
- As a visitor, I can filter properties by type.
- As a visitor, I can filter properties by bedrooms and bathrooms.

## Buy and Rent
- As a visitor, I can press Buy to view contact information.
- As a visitor, I can press Rent to view contact information.

## Design
- As a visitor, I can use a modern luxury interface.
